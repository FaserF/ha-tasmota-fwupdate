# ha-tasmota-fwupdate
 Clone of the official HA tasmota integration with extended firmware update functionality. You will need to disable the original Tasmota integration first! Otherwise errors about diplicates will occure.

 More details can be found (here in the open Pull Request)[https://github.com/emontnemery/hatasmota/pull/389] or (here in the community)[https://community.home-assistant.io/t/support-firmware-update-in-tasmota-integration/410167] and (here as a feature request)[https://github.com/emontnemery/hatasmota/issues/151].
